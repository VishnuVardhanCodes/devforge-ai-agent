# Generate Node.js REST API Project

When triggered:

"Build a simple Node.js REST API"

Perform the following actions:

Step 1 — Create Folder

Write directory:

demo-output/node-api/

---

Step 2 — Create package.json

Write file:

demo-output/node-api/package.json

Content:

{
"name": "node-api",
"version": "1.0.0",
"main": "src/index.js",
"scripts": {
"start": "node src/index.js"
},
"dependencies": {
"express": "^4.18.2"
}
}

---

Step 3 — Create src folder

Write directory:

demo-output/node-api/src/

---

Step 4 — Create index.js

Write file:

demo-output/node-api/src/index.js

Content:

const express = require("express");

const app = express();

app.use(express.json());

app.get("/", (req, res) => {
res.json({
message: "DevForge API running successfully"
});
});

const PORT = 3000;

app.listen(PORT, () => {
console.log(`Server running on port ${PORT}`);
});

---

Step 5 — Create README.md

Write file:

demo-output/node-api/README.md

Content:

# Node API Generated by DevForge

## Run Instructions

npm install
npm start

Server runs on:

http://localhost:3000

---

Step 6 — Create .env.example

Write file:

demo-output/node-api/.env.example

Content:

PORT=3000

---

Step 7 — Print Success Message

Print:

"Real Node.js API project generated successfully."

---
