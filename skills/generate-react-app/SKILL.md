# Generate React App Skill

You generate a complete React frontend project.

## Steps

1. Create project folder inside:

demo-output/react-app/

2. Run:

npx create-react-app demo-output/react-app

3. Create:

README.md

Inside:

Explain:

- How to install
- How to start
- Project description

4. Validate:

Ensure these exist:

- package.json
- src/App.js
- public/index.html

5. Print:

"React App generated successfully"
