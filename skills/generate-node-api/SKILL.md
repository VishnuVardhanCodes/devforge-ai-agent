# Generate Projects

When triggered, detect the project type based on the user prompt and generate real structured projects.

Do NOT create placeholder logic such as project.txt.

## Node.js REST API

When prompt contains:

"Build a simple Node.js REST API"

Run these Bash commands:

mkdir -p demo-output/node-api/src

---

Create package.json:

echo '{
"name": "node-api",
"version": "1.0.0",
"main": "src/index.js",
"scripts": {
"start": "node src/index.js"
},
"dependencies": {
"express": "^4.18.2"
}
}' > demo-output/node-api/package.json

---

Create index.js:

echo 'const express = require("express");

const app = express();

app.use(express.json());

app.get("/", (req, res) => {
res.json({
message: "DevForge API running successfully"
});
});

const PORT = process.env.PORT || 3000;

app.listen(PORT, () => {
console.log(`Server running on port ${PORT}`);
});' > demo-output/node-api/src/index.js

---

Create README.md:

echo '# Node API Generated by DevForge

## Installation

npm install

## Run

npm start

Server URL:

http://localhost:3000
' > demo-output/node-api/README.md

---

Create .env.example:

echo 'PORT=3000' > demo-output/node-api/.env.example

---

After execution:

Print:

"Real Node.js API project generated successfully."
